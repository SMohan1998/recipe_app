# Recipes App Backend

This is a backend REST API for managing recipes. It allows creating, reading, updating, and deleting recipes using Node.js, Express, and MongoDB.

## Features

- Create a new recipe
- Retrieve all recipes
- Retrieve a single recipe by ID
- Update a recipe by ID
- Delete a recipe by ID

## Technology Stack

- Node.js
- Express.js
- MongoDB (via Mongoose)
- Postman for API testing
